# SELECT in SQL

The `SELECT` statement retrieves rows from a table. You can filter, sort and aggregate results.

## Example

```sql
SELECT id, name FROM users WHERE id > 0 ORDER BY name;
```

## Output

```
id | name
1  | Alice
```

## Practice Questions

1. How do you select distinct values?
2. How do you limit the number of returned rows?
