# Loops in Python

Loops let you iterate over sequences or execute code multiple times. Common loops are `for` and `while`.

## Example

```python
for i in range(3):
    print('Iteration', i)

i = 0
while i < 3:
    print('While', i)
    i += 1
```

## Output

```
Iteration 0
Iteration 1
Iteration 2
While 0
While 1
While 2
```
